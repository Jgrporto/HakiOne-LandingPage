# HakiOne

Projeto frontend em Vite para rodar localmente.

## Como iniciar

1. Instale as dependencias:

```bash
npm install
```

2. Crie um arquivo `.env.local` com as variaveis do app:

```bash
VITE_APP_ID=your_app_id
VITE_APP_BASE_URL=your_backend_url
VITE_FUNCTIONS_VERSION=your_functions_version
```

3. Inicie o servidor local:

```bash
npm run dev
```

Por padrao, o Vite sobe em `http://localhost:5173`.
